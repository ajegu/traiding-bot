# Stratégies de Trading

## Stratégie RSI (Relative Strength Index)

Le RSI mesure la vitesse et l'ampleur des mouvements de prix.

### Configuration
- Période de calcul : 14 périodes
- Seuil de survente : RSI < 30
- Seuil de surachat : RSI > 70

### Signaux
- **Signal d'achat** : RSI < 30 (survente)
- **Signal de vente** : RSI > 70 (surachat)

## Stratégie Moyennes Mobiles (MA)

Les moyennes mobiles lissent les données de prix pour identifier les tendances.

### Configuration
- MA courte : 50 périodes (MA50)
- MA longue : 200 périodes (MA200)

### Signaux
- **Signal d'achat** : MA50 > MA200 (golden cross)
- **Signal de vente** : MA50 < MA200 (death cross)

## Indicateurs Supplémentaires (Futurs)

- **MACD** : Moving Average Convergence Divergence
- **Bollinger Bands** : Bandes de volatilité
- **Volume** : Confirmation des mouvements

## Ressources

- RSI : Mesure la vitesse et l'ampleur des mouvements de prix
- Moyennes Mobiles : Lissent les données de prix pour identifier les tendances
